# Welcome to what I will personally call my Quantum Journey.

I have studied my master in Quantum Information Science and Technologies. In there, I learned a lot; but I feel like it really wasn't enough. Maybe because I didn't take them as seriously as I should have or maybe because learning for the sake of passing didn't really allow me to actually LEARN.
So I decided to start self teaching myself a bit. Just looking for those places where the quantum computing world shines the brightest and seeing what I can get from them. 


I have decided to create a new Git-Hub entirely dedicated to this journey, which will pretty much record all of my learning process, work, maybe
even some circuits I eventually come up with, who knows. But I feel like having a journal where my progress can be seen, slowly rising from a newbie in the field like I feel right now even after my degree is almost done, to a fully fledged quantum engineer, is something I would like to have recorded somewhere.

The notebooks are based on courses I'm doing. The theory is all hand-written but the more practical part will be in the notebooks uploaded and updated. Everything will be in Spanish since that is my native tongue but if somehow anyone reaches this repository and wants to take some information feel free to do it and 
translate it for yourself or others, after all the courses are public so I am not doing anything new.

The README will be updated with the new content when it gets uploaded so it can work as an index. For now this is all there is.
